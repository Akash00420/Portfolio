Who I Am

I'm a full-stack JavaScript developer studying Computer Science at Narula Institute of Technology, Kolkata. I love turning complex problems into elegant, AI-powered web applications that real people can use.

My current focus is building myEnergy — an AI-powered electricity bill analyser for 300M+ Indian households, using the Claude Anthropic API for intelligent bill parsing and React for a seamless multilingual dashboard.

Outside of code, I compete in hackathons with my team Men Of Culture, where we build and ship production-ready applications under intense time pressure.
